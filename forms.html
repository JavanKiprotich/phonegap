HTML FORMS

HTML forms facilitates the user to enter data tha is sent to the server for processing. Can be used to collect data like contacts,logins, and personal details of users as well as feedback on users experience.

Tags used are form,input,textarea,select and option.

Form takes attributes method(how data is sent) and action(when data is to be sent)
Input has attribute of type="" which takes various types of inputs.

Tag description
form defines form to enter inputs
input defines input control
textarea allows text input
label defines a label for an input element
caption defines a caption for a fieldset
select defines a drop-down
option defines an option in a dropdown

input description
text standard textbox
checkbox toggling on and off,more than one can be selected
password displays asterisks
radio toggling,one at a go
file a file/image can be uploaded
number can input only integers
submit a button that submits form
email accepts only a valid email
date accepts date input
reset resets all form fields.
